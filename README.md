# Forum App

A Flutter-based Forum Application that allows users to register, log in, create discussion topics, and participate in forum conversations. The application uses Firebase Authentication for user management, Cloud Firestore for data storage, and Bloc State Management for handling application state.

## Features

### Authentication
- User Registration (Sign Up)
- User Login
- Firebase Authentication Integration
- Secure User Sessions

### Forum Management
- Create New Topics
- View Forum Topics
- Topic Details Screen
- Real-Time Data Storage with Firestore

### State Management
- Bloc Pattern Implementation
- Event-Based State Updates
- Clean Architecture Structure

## Technologies Used

- Flutter
- Dart
- Firebase Authentication
- Cloud Firestore
- Firebase Core
- Flutter Bloc
- Equatable
